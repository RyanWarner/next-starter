This is a [Next.js](https://nextjs.org/) project bootstrapped with [`create-next-app`](https://github.com/vercel/next.js/tree/canary/packages/create-next-app).

## Getting Started

Run the development server:

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

## What's inside

- TypeScript
- Styled Components
- Example components
- Cypress

## NVM

Using Node Version Manager, you can manage multiple Node versions and create a more consistent environment with other develepers and CI/CD. This project uses the most recent LTS version of Node.

After installing nvm,

```
nvm install
nvm use
```

See `.nvmrc`.
